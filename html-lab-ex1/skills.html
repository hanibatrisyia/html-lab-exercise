Glossary

Learn the most valuable web development terms to give your next project the head start it needs:

HTML
HyperText Markup Language: Standard language to create the structure and content of web pages.
CSS
Cascading Style Sheets: Language to describe the presentation, layout, colors and fonts of a document (styling).
JavaScript
Programming language that adds interactivity, animation, dynamic features to websites.
API
Application Programming Interface: A set of rules allowing different applications to communicate, often used to fetch data.
